# AI Accounting Assistant

This is an open-source AI-powered assistant designed to help users with accounting and taxation queries in India.

## Features
- GST guidance
- TDS compliance help
- Income tax support
- Financial query assistance

## Purpose
This project aims to help small businesses, CA students, and individuals by providing easy access to financial knowledge.

## Future Plans
- Integration with Claude AI API
- Website-based chatbot
- Multi-language support (English, Hindi, Marathi)
